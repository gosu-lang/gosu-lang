Upgrade Gosu code that was using a plain numeric expression in place of a interval in the for statement.
Gosu does not support numeric expressions in the for statement after the "in" keyword.
The code for(x in 10) {...} is illegal and will be upgraded with an interval like this: for(x in 0..|10) {...}

Note that under normal circumstances this inspection should be disabled as it was designed to be run just for the upgrade process.
